# ✦ Lumière OS — Güvenlik Politikası

## Güvenlik Açığı Bildirimi

Lumière OS'ta bir güvenlik açığı keşfettiyseniz, lütfen sorumlu bir şekilde bildirin.

### Bildirme Yöntemi

**Lütfen güvenlik açıklarını public GitHub Issues üzerinden PAYLAŞMAYIN.**

Bunun yerine:

1. **E-posta**: security@lumiere-os.org adresine şifreli e-posta gönderin
2. **GitHub Private Vulnerability Reporting**: Repository → Security → Report a vulnerability

### Bildirimi İçermesi Gerekenler

- Açığın açıklaması
- Tekrarlama adımları
- Potansiyel etki
- Varsa önerilen düzeltme

### Yanıt Süreci

| Aşama | Süre |
|---|---|
| İlk alındı bildirimi | 48 saat |
| Değerlendirme | 1 hafta |
| Düzeltme yayınlama | Ciddiyete göre 1–4 hafta |
| Kamuya açıklama | Düzeltme yayınlandıktan sonra |

### Desteklenen Sürümler

| Sürüm | Destek |
|---|---|
| v1.0.x | ✅ Aktif |
| < v1.0 | ❌ Desteklenmiyor |
