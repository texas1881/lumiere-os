# ✦ Lumière OS — Klavye Kısayolları

> `Super + /` ile bu rehberi masaüstünde açabilirsiniz.

---

## Temel İşlemler

| Kısayol | İşlev |
|---|---|
| `Super + Enter` | Terminal aç |
| `Super + D` | Uygulama başlatıcı (Rofi) |
| `Super + Q` | Pencereyi kapat |
| `Super + Shift + Q` | Oturumu sonlandır |
| `Super + L` | Ekranı kilitle |
| `Super + X` | Güç menüsü |
| `Super + /` | Kısayol rehberi |

## Pencere Yönetimi

| Kısayol | İşlev |
|---|---|
| `Super + F` | Tam ekran |
| `Super + Shift + F` | Monocle tam ekran |
| `Super + V` | Yüzen pencere modu |
| `Super + P` | Pseudotile |
| `Super + J` | Bölme yönünü değiştir |
| `Super + ← → ↑ ↓` | Pencere odağı taşı |
| `Super + Shift + ← → ↑ ↓` | Pencereyi taşı |
| `Super + Ctrl + ← → ↑ ↓` | Pencereyi boyutlandır |
| `Super + Fare Sol` | Pencereyi sürükle |
| `Super + Fare Sağ` | Pencereyi boyutlandır |

## Çalışma Alanları

| Kısayol | İşlev |
|---|---|
| `Super + 1-9` | Çalışma alanına git |
| `Super + Shift + 1-9` | Pencereyi çalışma alanına taşı |
| `Super + S` | Özel alan (scratchpad) |
| `Super + Shift + S` | Pencereyi özel alana taşı |
| `3 parmak kaydırma` | Çalışma alanı değiştir (touchpad) |

## Araçlar

| Kısayol | İşlev |
|---|---|
| `Super + E` | Dosya yöneticisi (Thunar) |
| `Super + W` | Duvar kağıdı seçici |
| `Super + T` | Tema değiştir (koyu ↔ açık) |
| `Super + I` | Ayarlar |
| `Super + M` | Sistem monitörü |
| `Super + Shift + V` | Pano geçmişi |

## Ekran Görüntüsü

| Kısayol | İşlev |
|---|---|
| `Print` | Alan seçerek ekran görüntüsü |
| `Shift + Print` | Tam ekran görüntüsü |
| `Super + Print` | Pencere görüntüsü |

## Multimedya

| Kısayol | İşlev |
|---|---|
| `Ses +` | Ses seviyesi artır |
| `Ses −` | Ses seviyesi azalt |
| `Ses Kapat` | Sessiz / sesli |
| `Mikrofon Kapat` | Mikrofon aç/kapat |
| `Parlaklık +` | Ekran parlaklığı artır |
| `Parlaklık −` | Ekran parlaklığı azalt |
